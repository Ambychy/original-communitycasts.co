Community Casts

$ ~ Community Casts is the place for developers and designers of all skill levels to find awesome, free educational screencasts about technology and design. It was built by Alex Booker and is 100% open source.

The home page shows a list of popular screencasts. If you want to view screencasts about a specific topic, click on a topic under the "Learn" header on the right-hand-side menu. If you want to submit a screencast, click "Submit".

Open Source

$ ~ Community Casts is 100% open source. I am always looking for people to help out. If you want to get involved, check out the open issues. You can always support the project by becoming a GitHub stargazer, too!

Community Casts is built using a bunch of cool open-source software. The main stack is Express (for the server), Angular (for the client), and MySQL (for the database).

Privacy

$ ~ Community Casts logs your IP address when you click a screencast. This so we can avoid counting the same view twice for ranking purposes.
