## `communitycasts.co/tags`

*Returns up to 20 tags. Most popular first.*

Sample response:

```json
{
  "tags": [
    {
      "tagName": "JavaScript",
      "count": 4
    },
    {
      "tagName": "Python",
      "count": 3
    },
    {
      "tagName": "Angular",
      "count": 2
    }
  ]
}
```
